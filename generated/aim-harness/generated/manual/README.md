# Manual Workspace

This directory is the reserved manual workspace for the generated `aim-harness`.

Current policy:

- manual flow is part of the default completion path
- writable target: `generated/manual/`
- local drafting workflow now exists in `skills/docs/manual-workflow/`
- prefer `generated/manual/<topic>/manual_draft.md` when the topic is stable
- external publish workflow is still deferred

Use this directory as the concrete destination for manual drafts created by `manual-workflow`.
